# 🚀 MapLite 说明 

这个项目融合了Mapbox 及 Openlayers 两种地图库操作，提供了一个简单的封装的兼容层，一套写法兼容两种地图库.
## 历史原因
原来的项目中出现了地图的过渡依赖Jquery,带有项目的副作用侵入性，所以``MapLite``在原有项目中提炼升级，不过渡依赖Jquery。
## 目录说明

### 文件夹：baseMap
- baseMap.js
> 在全局暴露地图操作对象

- map.proxy.js
> 胶水层的地图代码，主要的核心逻辑层

### 文件夹: event
> 内部的一些核心事件，例如地图加载完成事件，会派发``MapLoaded``事件到全局

### 文件夹: layer
- layer.js
> 提供了简易的图层封装，内部实现依赖 ``map.proxy.js``

### setting 文件夹

- setting.js
> 配置文件，配置地图使用哪种地图库及对应地图库初始化地图的配置
